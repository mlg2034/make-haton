
## Features

All code analysis rules in this package

## Usage

1. Add to `dev_dependency` 

```yaml
    flutter_lints: ^2.0.0
    dart_code_metrics: ^5.6.0
    taw_linters:
      path: ../packages/taw_linters
```

2. Update your `analysis_options.yaml`

```yaml
include: package:taw_linters/linters.yaml
```
